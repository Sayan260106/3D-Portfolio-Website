# SayanOS — Interactive 3D Portfolio

A futuristic **3D desktop-inspired portfolio website** built with modern web technologies, inspired by a full operating system experience inside the browser.

The project combines:

* Interactive 3D environments
* Desktop-style window management
* Smooth animations
* Draggable applications
* Realistic workspace aesthetics
* Responsive modern UI

Designed and developed to create a unique developer portfolio experience.

## 3D Interactive Environment

* Immersive 3D room setup
* Realistic lighting and shadows

* Multi-window management

# Installation & Setup

## Clone the repository

```bash
git clone https://github.com/your-username/sayanos.git
```

## Navigate to project directory

```bash
cd sayanos
```

## Install dependencies

```bash
npm install
```

## Start development server

```bash
npm run dev
```

---

# Build for Production

```bash
npm run build
```

---

# Preview Production Build

```bash
npm run preview
```

---

# Environment Notes

Large 3D model files are excluded using `.gitignore` to keep the repository lightweight.

If assets are missing after cloning:

* Add required `.glb` model files inside the `public/models` directory.
* Ensure textures and assets paths are correct.
